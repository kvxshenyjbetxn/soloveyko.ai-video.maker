# pollinations(поллінейшн) - сервіс для генерації зображень

- це сервіс для генерації зображень, з безкоштовними моделями.

- Налаштування Pollinations знаходяться за шляхом Налаштування - API - Зображення - Pollinations.

- швидкість генерації залежить від сервісу а не від нашої програми.

- сервіс мае дуже суворі обмеження щодо кількості генерації, а саме рейт ліміти. якщо ви не впишете свій апі ключ, то сервіс буде працювати дуже повільно. для пришвидшення потрібно отримати безкоштовний апі ключ на сайті сервісу https://enter.pollinations.ai/ та вписати його у відповідне поле в налаштуваннях програми. Налаштування - API - Зображення - Pollinations - API ключ. але навіть маючи апі ключ, сервіс все одно працюе набагато повільніше за інші сервіси, так як є безкоштовним, якість також відповідна, без ключа вам доступно всього лише декілька моделей дуже низької якості, отримавши ключ, ви можете використовувати більший список вже якісних моделей.

- для отримання апі ключа потрібно на сайті сервісу https://enter.pollinations.ai/  зарееструватись за допомогою github акаунту, після чого отримати апі ключ.

- ми не рекомендуемо використовувати данний сервіс, так як він працюе дуже повільно і мае низьку якість генерації з деякими моделями, краще якщо безкоштовно то використати сервіс elevenlabsimage, він є повністью безкоштовним вам всього лише потрібно отримати безкоштовний апі ключ в боті  сервісу elevenlabsimage, ось посилання на бота https://t.me/elevenLabsVoicerBot
